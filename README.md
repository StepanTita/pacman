## üöÄ Getting Started:

### Prerequisites:

* **python** >= 3.6

Install required packages using the following command:

```bash
pip install -r requirements.txt
```

### Running the Game:
1. Clone the repository:
```bash
git clone https://github.com/StepanTita/pacman.git
```
2. Navigate to directory pacman:
```bash
cd pacman
```

3. Run the game:
```bash
python main/run.py
```
